2.4.1.2 Scenario 1 Answer Key

Objectives

· Configure hostnames and IP addresses on two Cisco Internetwork Operating System (IOS) switches using the command-line interface (CLI).
· Use Cisco IOS commands to specify or limit access to the device configurations.
· Use IOS commands to save the running configuration.
· Configure two host devices with IP addresses.
· Verify connectivity between the two PC end devices.
Scenario

As a recently hired LAN technician, your network Student-1has asked you to demonstrate your ability to configure a small LAN. Your tasks include configuring initial settings on two switches using the Cisco IOS and configuring IP address parameters on host devices to provide end-to-end connectivity. You are to use two switches and two hosts/PCs on a cabled and powered network.

Requirements

· Use a console connection to access each switch.
· Name Class-Aand Class-Bswitches.
· Use the xAw6k password for all lines.
· Use the 6EBUp secret password.
· Encrypt all clear text passwords.
· Include the word warning in the message-of-the-day (MOTD) Banner.
· Configure addressing for all devices according to the Addressing Table.
· Save your configurations.
· Verify connectivity between all devices.

Answers

· On the bottom lefthand side of your screen, click the little lightning icon. Then click the console cable (second from the left) and click on the PC labeled Manager. Click RS232 and then click on the Class-ASwitch. Select console form the drop-down menu. 

· On the bottom lefthand side of your screen, click the little lightning icon. Then click the console cable (second from the left) and click on the PC labeled Student-2. Click RS232 and then click on the Class-BSwitch. Select console form the drop-down menu. 

· Click on Student-1-> Desktop -> Terminal -> OK

· Type in the following commands.

Switch>en
Switch#conf t
Switch(config)#hostname Class-A
Class-A(config)#line con 0
Class-A(config-line)#password xAw6k
Class-A(config-line)#line vty 0 15
Class-A(config-line)#password xAw6k
Class-A(config-line)#exit
Class-A(config)#enable secret 6EBUp
Class-A(config)#service password-encryption
Class-A(config)#banner motd "Warning! Keep Out!"
Class-A(config)#do writ
Building configuration...
[OK]
Class-A(config)#int vlan 1
Class-A(config-if)#ip address 172.16.5.35 255.255.255.0
Class-A(config-if)#no shut

Class-A(config-if)#
%LINK-5-CHANGED: Interface Vlan1, changed state to up

%LINEPROTO-5-UPDOWN: Line protocol on Interface Vlan1, changed state to up

· Click Exit

· Click on IP-Configuration

· In the IP Address Field type 172.16.5.50 

· In the Subnet Mask Feild type 255.255.255.0

· Return to the Network Topology 

· Click on Student-2 -> Desktop -> Terminal -> OK

· Type in the following commands.

Switch>en
Switch#conf t
Switch(config)#hostname Class-A
Class-B(config)#line con 0
Class-B(config-line)#password xAw6k
Class-B(config-line)#line vty 0 15
Class-B(config-line)#password xAw6k
Class-B(config-line)#exit
Class-B(config)#enable secret 6EBUp
Class-B(config)#service password-encryption
Class-B(config)#banner motd "Warning! Keep Out!"
Class-B(config)#do writ
Building configuration...
[OK]
Class-B(config)#int vlan 1
Class-B(config-if)#ip address 172.16.5.40 255.255.255.0
Class-B(config-if)#no shut

Class-B(config-if)#
%LINK-5-CHANGED: Interface Vlan1, changed state to up

%LINEPROTO-5-UPDOWN: Line protocol on Interface Vlan1, changed state to up

· Click Exit

· Click on IP-Configuration

· In the IP Address Field type 172.16.5.60 

· In the Subnet Mask Feild type 255.255.255.0

2.4.1.2 Scenario 3 Answer Key

Objectives

· Configure hostnames and IP addresses on two Cisco Internetwork Operating System (IOS) switches using the command-line interface (CLI).
· Use Cisco IOS commands to specify or limit access to the device configurations.
· Use IOS commands to save the running configuration.
· Configure two host devices with IP addresses.
· Verify connectivity between the two PC end devices.
Scenario

As a recently hired LAN technician, your network User-01 has asked you to demonstrate your ability to configure a small LAN. Your tasks include configuring initial settings on two switches using the Cisco IOS and configuring IP address parameters on host devices to provide end-to-end connectivity. You are to use two switches and two hosts/PCs on a cabled and powered network.

Requirements

· Use a console connection to access each switch.
· Name ASw-1 and ASw-2 switches.
· Use the xAw6k password for all lines.
· Use the 6EBUp secret password.
· Encrypt all clear text passwords.
· Include the word warning in the message-of-the-day (MOTD) Banner.
· Configure addressing for all devices according to the Addressing Table.
· Save your configurations.
· Verify connectivity between all devices.

Answers

· On the bottom lefthand side of your screen, click the little lightning icon. Then click the console cable (second from the left) and click on the PC labeled User-01. Click RS232 and then click on the ASw-1 Switch. Select console form the drop-down menu. 

· On the bottom lefthand side of your screen, click the little lightning icon. Then click the console cable (second from the left) and click on the PC labeled User-02. Click RS232 and then click on the ASw-2 Switch. Select console form the drop-down menu. 

· Click on User-01 -> Desktop -> Terminal -> OK

· Type in the following commands.

Switch>en
Switch#conf t
Switch(config)#hostname ASw-1
ASw-1(config)#line con 0
ASw-1(config-line)#password xAw6k
ASw-1(config-line)#line vty 0 15
ASw-1(config-line)#password xAw6k
ASw-1(config-line)#exit
ASw-1(config)#enable secret 6EBUp
ASw-1(config)#service password-encryption
ASw-1(config)#banner motd "Warning! Keep Out!"
ASw-1(config)#do writ
Building configuration...
[OK]
ASw-1(config)#int vlan 1
ASw-1(config-if)#ip address 172.16.5.35 255.255.255.0
ASw-1(config-if)#no shut

ASw-1(config-if)#
%LINK-5-CHANGED: Interface Vlan1, changed state to up

%LINEPROTO-5-UPDOWN: Line protocol on Interface Vlan1, changed state to up

· Click Exit

· Click on IP-Configuration

· In the IP Address Field type 172.16.5.50 

· In the Subnet Mask Feild type 255.255.255.0

· Return to the Network Topology 

· Click on User-02 -> Desktop -> Terminal -> OK

· Type in the following commands.

Switch>en
Switch#conf t
Switch(config)#hostname ASw-1
ASw-2(config)#line con 0
ASw-2(config-line)#password xAw6k
ASw-2(config-line)#line vty 0 15
ASw-2(config-line)#password xAw6k
ASw-2(config-line)#exit
ASw-2(config)#enable secret 6EBUp
ASw-2(config)#service password-encryption
ASw-2(config)#banner motd "Warning! Keep Out!"
ASw-2(config)#do writ
Building configuration...
[OK]
ASw-2(config)#int vlan 1
ASw-2(config-if)#ip address 172.16.5.40 255.255.255.0
ASw-2(config-if)#no shut

ASw-2(config-if)#
%LINK-5-CHANGED: Interface Vlan1, changed state to up

%LINEPROTO-5-UPDOWN: Line protocol on Interface Vlan1, changed state to up

· Click Exit

· Click on IP-Configuration

· In the IP Address Field type 176.16.5.50 

· In the Subnet Mask Feild type 255.255.255.0
